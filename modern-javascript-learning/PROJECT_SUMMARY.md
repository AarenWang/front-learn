# 现代 JavaScript 学习项目概览

## 🎉 项目初始化完成

基于 README 中的 16 课时学习蓝图，项目已经构建好交互式学习门户，帮助你系统掌握现代 JavaScript 与工程化能力。

## ✅ 已完成的功能

### 1. 项目基础配置
- ✅ 使用 Vite 初始化 React + TypeScript 应用
- ✅ 配置 pnpm、ESLint、Prettier、Tailwind CSS
- ✅ 设置路径别名 `@/*` 指向 `src/*`
- ✅ 配置 Vitest 测试环境和 jsdom

### 2. 目录结构
```
src/
├── app/                 # 路由入口 (App.tsx)
├── components/          # 通用组件 (Button/Card/ThemeToggle 等)
├── pages/               # 页面组件 (首页 + 课时详情 + 404)
├── styles/              # 全局样式
├── types/               # 类型定义与课时数据
└── test/                # 测试配置
```

### 3. 课时体验
- ✅ 首页展示 16 个课时的基础知识、项目实战与课时时长
- ✅ 动态路由 `/lessons/:lessonSlug` 显示课时目标、项目交付物、实践任务与验收标准
- ✅ 暗色模式、自适应布局、进阶资源链接

### 4. 工程能力
- ✅ Tailwind 设计体系 + 自定义组件库
- ✅ 统一按钮、卡片、主题切换组件
- ✅ 示例测试：Button 组件单测
- ✅ 可扩展的课时数据结构，便于后续拓展

## 🚀 如何使用

```bash
cd modern-javascript-learning
pnpm install
pnpm dev
```
访问 `http://localhost:5173` 即可体验交互式学习网页。

## 📚 学习路径亮点

- **基础知识**：语法、函数、对象、模块化、DOM、异步
- **工程化**：包管理、构建工具、质量保障、自动化测试
- **项目实战**：每课时配套项目交付物与验收标准
- **进阶能力**：性能优化、监控、部署与 CI/CD

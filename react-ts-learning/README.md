# React + TypeScript 学习项目

这是一个基于 AI Vibing Code 的 React + TypeScript 学习项目，包含完整的学习路径和实战演示。

## 🚀 快速开始

### 安装依赖

```bash
pnpm install
```

### 启动开发服务器

```bash
pnpm dev
```

### 构建项目

```bash
pnpm build
```

### 运行测试

```bash
pnpm test
```

### 代码格式化

```bash
pnpm format
```

## 📚 学习路径

本项目包含 12 个学习阶段（S0-S11），每个阶段都有：

- 📖 **学习目标** - 明确的学习重点
- ✅ **任务清单** - 具体的实践任务
- 🎯 **验收标准** - 完成标准
- 💻 **演示项目** - 可交互的代码示例
- 📝 **代码要点** - 关键知识点解释

### 学习阶段概览

| 阶段 | 主题 | 主要内容 |
|------|------|----------|
| S0 | 基础起步 | JSX、组件、Props/State、单向数据流 |
| S1 | Hooks 核心 | useState、useEffect、useMemo、useCallback |
| S2 | 事件与表单 | ChangeEvent、受控组件、表单验证 |
| S3 | 路由 | React Router、嵌套路由、懒加载 |
| S4 | 数据获取 | Axios、React Query、数据缓存 |
| S5 | 全局状态 | Context API、Zustand、状态管理 |
| S6 | 表单进阶 | React Hook Form、Zod 验证 |
| S7 | 样式体系 | Tailwind CSS、响应式设计、主题 |
| S8 | 测试 | Vitest、React Testing Library |
| S9 | 性能与可访问性 | 懒加载、虚拟滚动、ARIA |
| S10 | 部署与 CI | Vercel 部署、GitHub Actions |
| S11 | 进阶路线 | Next.js、SSR、Server Components |

## 🛠️ 技术栈

- **框架**: React 19 + TypeScript
- **构建工具**: Vite
- **路由**: React Router v7
- **样式**: Tailwind CSS
- **状态管理**: Zustand
- **数据获取**: TanStack Query (React Query)
- **表单处理**: React Hook Form + Zod
- **测试**: Vitest + React Testing Library
- **代码质量**: ESLint + Prettier

## 📁 项目结构

```
src/
├── app/                 # 应用入口
├── components/          # 通用组件
├── features/           # 功能模块
├── hooks/              # 自定义 Hooks
├── pages/              # 页面组件
├── services/           # API 服务
├── store/              # 状态管理
├── styles/             # 样式文件
├── types/              # TypeScript 类型定义
├── utils/              # 工具函数
└── test/               # 测试文件
```
